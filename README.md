## Setting
- **.env**  
  Setting your API for Cloud Google, Midtrans, and email account in the `.env` file.
- **Database**  
  Configure the MySQL database in `database.js`.
- **Port**  
  Set the port in `app.js`.
  
## Run Script
- `npm install`
- `npm start`
